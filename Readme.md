# 🔐 Login Page UI (Split Layout)

This project is a responsive and visually appealing **Login Page User Interface** built using HTML and CSS. The layout follows a split-screen design with an illustration on one side and a login form on the other.

---

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in any modern browser
3. Ensure the image path is correctly set inside the `img` folder

---

## 🚀 Features

- Split layout design (left: image, right: form)
- Full-height container (`100vh`)
- Centered content using Flexbox
- Clean and structured form layout
- Styled input fields with hover and focus effects
- Modern button design with interaction feedback
- Background image applied using CSS for better control

---

## ⚙️ Technologies Used

- HTML5
- CSS3 (Flexbox)
